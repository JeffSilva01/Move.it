- [VSCode](https://code.visualstudio.com)

<h2 id="usage" > 👷 Usage </h2>

Required! Install git, node and yarn (or npm).

```bash
# Clone Repository
$ git clone https://github.com/JeffSilva01/Move.it.git

# Go to server folder
$ cd Move.it

# Install Dependencies
$ yarn

# Run Aplication
$ yarn dev

# Access localhost
http://localhost:3000
```
